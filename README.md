## 曾经的我们:

* 岁月如梭，光阴似箭；往事难忘，温馨如昔。转眼间，你我已经毕业5年了。亲爱的你，是否还记得0502083XX那个专属于你的代号？是否还记得大东湖畔美丽的校园,那个曾经留下过你轻快的脚步、欢乐笑语的地方？
